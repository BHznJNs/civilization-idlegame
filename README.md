# 使用方法

本项目已 commit 构建后的代码，你可以在下载源码后直接在浏览器中打开 `dist/` 中的 `index.html` 使用。

若需要自行修改及构建，请查看下面命令

安装依赖
```
npm install
```

启用 `vue-cli-service` (在浏览器中输入 http://localhost:8080 打开):
```
npm run serve
```

构建项目:
```
npm run build
```
