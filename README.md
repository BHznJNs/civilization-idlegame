# 文明：放置游戏 Civilization-IdleGame

学习 [Vue.js](https://vue3js.cn/) 时写的小游戏。  
尚未开发完成。

* * *

# 使用方法

启用 `vue-cli-service` (在浏览器中输入 `http://localhost:8080` 打开):
```
npm run serve
```

构建项目:
```
npm run build
```
* * *
